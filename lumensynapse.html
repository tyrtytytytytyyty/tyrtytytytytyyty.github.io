Obsidian Plugin

Lumen Synapse — Agent Activity Monitor

Video coming soon

An Obsidian plugin that makes a personal AI agent's vault activity visible in real time. When my AI agents access files, graph nodes pulse with their assigned color, the file explorer glows on the active row, and connections render as live synapses. A transparency layer for an agentic personal knowledge system.

The pulse stream reads from JSONL log files written by each agent's PostToolUse hook. The plugin watches mtime diffs, surfaces new events, and animates them on the graph and explorer. Built as a productivity layer first and a creative one second — knowing what your agents are doing in real time changes how you collaborate with them.
