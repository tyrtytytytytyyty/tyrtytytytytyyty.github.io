Character Shop UI

Screenshot coming soon

A modular Roblox shop interface built on commission. Category tabs, item grid with hover states, visibility toggles for owned items, and UIStroke-based selection highlights with a yellow accent.

Designed the component architecture and shipped the interaction layer as a reusable LocalScript module — the shop reads its catalog from a ModuleScript, so swapping in new items doesn't touch any UI code.
