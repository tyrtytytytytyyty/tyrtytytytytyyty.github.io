Deck System

Screenshot coming soon

A content-agnostic deck loader. Any subject can be dropped in — define a JSON deck with MCQs and FRQs, and the engine handles the rest. Built decks so far: AP World History, AP Precalc.

The deck format separates content from engine completely, so new subjects don't touch any code. Drill scales linearly with effort — adding a new deck is content work, not engineering work.
